# Y Combinator Product Messaging Skill

## Overview
This skill helps AI generate concise, user-focused product messaging following Y Combinator principles. Focus on value, clarity, simplicity, and actionable next steps.

## Principles

### 1. Value First
State the user value clearly. Avoid technical jargon or infrastructure details.  
*Example:* "Provides AI support during phone calls" (value).  
*Not:* "GPT-based call automation" (technology).

### 2. How It Works
Explain how the product delivers value in clear steps without unnecessary technical details.  
*Example:*  
1. User creates an account  
2. AI answers calls on behalf of the user  
3. System generates a summary report

### 3. Clear Next Step
Tell users exactly what to do next: "Sign up", "Try now", "Request demo".

### 4. Keep It Simple
Use short sentences and plain language. Users should understand your message within 5 seconds.

### 5. Test and Iterate
Try different messaging variations, collect user feedback, and optimize continuously.

## Messaging Structure

- **Problem:** State the user's pain clearly and concisely.  
- **Solution:** Show how the product solves this problem.  
- **Value:** Explain the benefit the user gets.  
- **Next Step:** Indicate the immediate user action.

## AI Prompt Example

You are an AI assistant specialized in startup messaging based on YC principles.
Input: product description + target user
Output: simple value statement, "how it works" bullets, and clear user action
Constraints: use plain language, no jargon, keep it concise


## Fail Cases
- Focusing only on technology  
- Using complex or obscure terms  
- No clear user action provided

## Success Indicators
- Users can restate the product value in their own words  
- Reduced bounce rates on landing pages  
- Higher conversion on first user action
